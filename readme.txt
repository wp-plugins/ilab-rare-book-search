This plugin enables your visitors to search for old or rare books for sale online by over 1100 of the world’s leading antiquarian booksellers.

== Description ==

The plugin installs a simple sidebar search form  that can be used to find old, rare and second-hand books for sale by affiliates of the [International League Of Antiquarian Booksellers](http://www.ilab.org). 

= Features =

- A simple search for Books by Author, Title or Keyword is available from any page where the sidebar is displayed.
- An Advanced Search form also permits searches that limit are limited to a specified date range or price range.
- Prices can be shown in any of 12 different currencies.
- Search results can be sorted by publication date, price, author or title.
- Each result includes a full description of the book, with photos when available, and links directly to a source page where the book can be purchased.

This search utility is sponsored by the International League of Antiquarian Booksellers.  Founded in 1948 with the purpose of increasing contact and cooperation among booksellers around the world, the League now represents nearly 1800 of the world’s leading rare book dealers who are members of the 22 national associations that comprise the ILAB.  Membership is governed by strict standards for professional experience and ethics.

== Installation ==

1. You can download and install this plugin using the built in WordPress plugin installer. If you download it manually, make sure it is uploaded to "/wp-content/plugins/ilab-plugin/".
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Add the search widget to your sidebar or footer using the 'Appearance' -> 'Widgets' menu.
